# Web Scrapping

This is a simple web scrapping project that reads a table of invoices and generate a csv file with dueDate to import at Google Agenda.

## Techs

 - node.js
 - nodemon 
 - puppeteer
 - csv-writer


## Getting Start

setup your User and Password at config.js file

```js
clone repo  
cd agenda  
npm i  
npm run dev
```
